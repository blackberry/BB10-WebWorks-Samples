swipemenu.js

A swipe-down menu framework for BlackBerry WebWorks SDK for BB10.

Designed to be easy to use (JavaScript only; no CSS or HTML), simply follow these steps to add a swipe-down menu bar to your PlayBook application:

  1. Add feature to config.xml:
    <feature id="blackberry.app" required="true" version="1.0.0.0"/>
  2. Include swipemenu.js:
    <script src="swipemenu.js"></script>
  3. Add buttons for your menu during page onload event:
    swipemenu.addButton("Label", doButton, false, "img.png"); // title, callback, alignRight, imgPath
